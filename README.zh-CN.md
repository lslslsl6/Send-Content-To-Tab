# 发送内容到标签页 (Send Content to Tab)

一款可将网页文本或链接内容发送到侧边栏，编辑后在新标签页或无痕式标签页中打开的扩展。

## 概述

**发送内容到标签页** 是一款浏览器扩展，提供便捷的侧边栏面板，用于快速处理网页中的文本内容。无论是搜索选中的文本、访问链接、获取当前页面网址，还是在无痕窗口中打开内容——所有这些都可以直接从侧边栏完成，无需离开当前页面。

# 功能特性

- **发送到侧边栏** — 通过右键菜单或快捷键，将选中文本、链接或当前页面网址发送到侧边栏。
- **编辑与处理** — 在侧边栏中直接编辑文本、清空内容或获取当前页面网址。
- **打开标签页** — 在新标签页或无痕式标签页中打开处理后的内容。智能 URL 识别：网址直接打开，其他文本通过搜索引擎搜索。
- **文本暂存** — 可选在侧边栏中保存和加载文本内容。

## 权限说明

| 权限             | 用途                     |
| ---------------- | ------------------------ |
| `sidePanel`    | 显示侧边栏界面           |
| `contextMenus` | 添加右键菜单项           |
| `storage`      | 保存设置和暂存文本       |
| `tabs`         | 获取当前标签页 URL       |
| `search`       | 调用浏览器设置的搜索引擎 |

## 隐私政策

本扩展 **不会** 收集、存储或传输任何个人数据。所有内容处理均在浏览器本地完成。不包含任何分析、跟踪或外部网络请求。
